老年人合理用药干预 · 药师工作台（离线版 V1.1）使用说明
====================================================

一、这是什么
配合《研究病例记录表》V2.0 和《附件2 干预措施方案及标准化话术》使用的药师工具，
用于：电子量表自动计分分层、自动生成干预处方、随访提醒与录入、话术一键复制、数据导出。

二、如何使用（无需安装、无需联网）
1. 把"老年人合理用药干预药师工作台.html"这一个文件发给各分中心（微信/U盘均可）。
2. 用电脑或平板的浏览器（推荐 Edge / Chrome）双击打开即可使用。
3. 首次使用：进入【设置】填写中心号、中心名称、药师姓名电话、群推送起始日期。

三、日常工作流
入组（7步，系统与流程一一对应，每步已内嵌对应话术，可展开查看、可跳过）：
  ① 知情同意：A1 话术告知 → 患者在签字板电子签名（或勾选"已签纸质版"），
     电子签字页永久保存在患者档案，可随时查看、打印；
  ② 基本信息登记；
  ③ 用药清单：按 药名/剂量/频次/备注 逐行录入（＋/－增减行），
     也可直接给患者的药盒、纸质药单拍照存档（自动压缩，仅存本机，最多5张）；
  ④ AMRS 依从性 12 题（A2 话术引导）；
  ⑤ 药物素养量表 41 题（系统自动处理全部反向题计分、自动分层）；
  ⑥ 结果与干预处方：自动生成干预处方（B1/B2/B3 话术一键复制），
     并对 41 条逐条给出干预建议（1~2分强力干预、3~4分普通干预、5分不干预），
     同屏生成"患者结果二维码"——请患者手机拍下二维码，回家微信扫一扫
     即可查看大字号的个人结果与可执行建议（不含姓名等身份信息）；
  ⑦ 入群邀请（A3 话术）→ 保存。
平时：工作台首页会显示"本周该发第几期群推送"，一键复制到微信群；
      群内答疑可在【话术库】搜索 Q1~Q8 建议回复。
随访：首页"随访待办"自动列出进入 1个月/3个月 窗口期（±7天）的患者，
      点"录入随访"按 CRF 条目顺序引导填写；3个月随访含量表复测与满意度，自动对比基线。

四、数据安全（重要）
· 数据只保存在本机浏览器中，不上传任何服务器；
· 系统内只录入受试者编号和姓名缩写，不含身份证号等直接身份信息；
· 请每周在【数据导出】中导出一次 JSON 备份保存到工作电脑；
· 按项目要求定期导出 CSV 汇总表上报项目组（Excel 可直接打开，可导入 SPSS）；
· 不要清除浏览器"网站数据/缓存"，否则本机数据会丢失（有备份可恢复）。

五、计分口径（与 CRF V2.0 一致，系统自动执行）
· 素养量表 41 题，总分 41–205，平均分=总分/41；
· 平均分≥4 良好、3~4 中等、≤3 不足；总分≥157 或平均分≥3.82 为达标；
· 全部反向题（含 AMRS 第12题）由系统自动赋值，药师按患者原话选择即可。

六、AI 智能辅助（通义千问，可选）
已接入阿里云通义千问，为药师提供三项"草稿"建议，位于顶部【AI辅助】页：
  ① 群答疑草稿生成：粘贴患者提问，生成一段可发群/私聊的通俗回复草稿；
  ② 随访记录智能摘要：输入随访通话要点，整理为 CRF 格式随访记录草稿；
  ③ 用药清单相互作用初筛：输入或从患者一键带入用药清单，做重复用药/相互作用初筛提示。
启用方法：在【设置→AI 智能辅助】填写千问 API 密钥（阿里云百炼 bailian.console.aliyun.com
  创建 sk- 开头的 API-KEY），勾选"启用"并保存，可点"测试连接"验证。
重要提示：
  · API 密钥只保存在本机浏览器，不写入网页、不含在 JSON 备份和 CSV 导出中；怀疑泄露
    请到阿里云百炼控制台停用重建；
  · 所有 AI 输出均为草稿，须经药师审核、按项目话术纪律修改后再使用，系统不会自动发送；
  · 请勿在输入中包含患者姓名、身份证号、电话等身份信息；相互作用初筛仅为初筛，
    最终以药品说明书和药师/医师审方为准；
  · 计分、分层、话术匹配仍为确定性规则，不依赖 AI，不启用 AI 亦可正常使用全部核心功能。

技术支持：项目组（版本 2026-07-14）
